# 📈 Stock Trading Platform

A modern Stock Trading Platform built using React, TypeScript, Vite, and Tailwind CSS. This application simulates stock market trading, allowing users to view stock prices, buy and sell shares, track portfolio performance, and manage transactions in a user-friendly interface.

## 🚀 Features

* User-friendly trading dashboard
* View available stocks and market data
* Buy and sell stocks
* Portfolio management
* Transaction history tracking
* Real-time UI updates
* Responsive design
* Modern and clean interface
* Built using Object-Oriented Programming (OOP) concepts

## 🛠️ Technologies Used

### Frontend

* React
* TypeScript
* Vite
* Tailwind CSS

### Concepts

* Object-Oriented Programming (OOP)
* Component-Based Architecture
* State Management
* Responsive Web Design

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/mpreethamteja-star/stock-trading-platform.git
```

### Navigate to the Project Directory

```bash
cd stock-trading-platform/project
```

### Install Dependencies

```bash
npm install
```

### Run the Development Server

```bash
npm run dev
```

### Open in Browser

```text
http://localhost:5173
```
